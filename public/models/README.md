# Avatar asset

- Runtime model: `ecy-avatar.glb`
- Fallback preview: `ecy-avatar-preview.png`
- Source archive: maintained outside the code repository in the project's local asset storage (`项目资料/虚拟分身模型`)
- Export pipeline: `tools/blender_avatar_pipeline.py`
- Blender version used: 5.2 LTS

The model is currently a private prototype asset supplied by the project owner. Redistribution and commercial licensing must be confirmed before any public release. Do not replace the runtime files without running `pnpm test:assets` and recording the new source and license.
